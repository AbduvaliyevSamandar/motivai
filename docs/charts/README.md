## Diplom matniga kiritish

GitHub'da bu papkani ochsangiz, har bir mermaid blok rasm sifatida render
bo'ladi. Diplom matnida ko'chirib joylash uchun:

1. Faylni GitHub'da oching
2. Diagramma ustiga sichqonchani olib boring → "Copy as image" yoki
   o'ng tugma → "Save image as..."
3. PNG sifatida kompyuterga saqlang
4. Word/LaTeX hujjatga kiriting

Yoki [Mermaid Live Editor](https://mermaid.live) ga ko'chirib, SVG/PNG
sifatida eksport qiling — yuqori sifatli vektor.
